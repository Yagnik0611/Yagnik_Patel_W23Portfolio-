Recipe management system - Java Spring Boot

The project at hand is a recipe management system developed using the Java programming language and the Spring Boot framework for the backend, with React used for the frontend. The system provides functionalities such as recipe addition, deletion, and updating, as well as meal creation and planning on a date-wise basis. Users are able to view and mark meals as favorites.

The system is designed to handle complex database operations and interactions, with robust error handling mechanisms in place to ensure data integrity and reliability. The use of industry-standard programming practices such as dependency injection, MVC architecture, and RESTful API design principles ensures optimal system performance and scalability. Overall, this project represents a sophisticated and comprehensive solution for recipe management and meal planning.

Weather Website - React

My weather application is a web-based platform that allows users to view live weather data from different locations across the world. It is built using the MERN (MongoDB, Express, React, Node) stack, and utilizes various APIs to fetch real-time weather data. The application features an intuitive user interface that displays current weather conditions, including temperature, wind speed, humidity, and pressure. Users can also view hourly and daily weather forecasts, as well as historical data for different locations. The application provides users with detailed information about weather patterns, precipitation, and other weather-related phenomena.

The application fetches data from a variety of sources, including the OpenWeather API, and uses advanced algorithms to process the data and generate accurate weather forecasts. The back-end of the application is built using Express and Node.js, while the front-end is built using React. The application also utilizes MongoDB to store user data and weather information.

Park and Recreation - Capstone Project

My capstone project is aimed at creating a website that offers a unique platform for park owners to showcase their facilities and services to the public. Through our website, park owners can easily add and update park and facility information, while users can access this information and make bookings for their preferred facilities. Our website also has an admin section where park owners can keep track of their clients' information and ensure that all details are up-to-date.
